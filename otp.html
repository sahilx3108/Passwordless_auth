One-Time Password (OTP)

OTP authentication uses short-lived codes sent to a user's phone or email. It is simple to implement and familiar to most users.

  • Pros: Easy to implement, user familiarity
  • Cons: Vulnerable to phishing and SIM swapping attacks
  • Use Cases: Login verification, password recovery
  • Security Notes: Use rate limiting, avoid using only SMS
